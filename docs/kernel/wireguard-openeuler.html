    Ryan 7/10/2023

    • 参考wireguard官方编译文档

sudo make -C wireguard-linux-compat/src install

    # 错误LINUX_VERSION_CODE >= KERNEL_VERSION(5, 6, 0)

    这个错误其实是因为在linux内核5.6以后，已经默认加上了wireguard的模块，官方警告我们不需要自己再构建了。

    然而我们是在其他开源版本linux修改后的欧拉上编译，所以这个警告我们需要给他去掉。

    这个文件就是wireguard-linux-compat/src/compat/compat.h

    把里面这些内容删掉就ok了

    -#if LINUX_VERSION_CODE >= KERNEL_VERSION(5, 6, 0)
-#error "WireGuard has been merged into Linux >= 5.6 and therefore this compatibility module is no longer required."
-#endif

    可以看到wireguard.ko依赖udp_tunnel,ip6_udp_tunnel这两个内核模块，先挂载他俩，在挂载wireguard就可以了。

    modprobe udp_tunnel
modprobe ip6_udp_tunnel
modprobe wireguard
